# jQuery Notes

## 📌 Prerequisites
Before learning jQuery, you should know:
- HTML
- CSS
- JavaScript

---

## 📖 What is jQuery?

jQuery is a lightweight JavaScript library.

It follows the principle:

> `"Write less, do more"`

jQuery makes it easier to use JavaScript on websites.

It reduces long JavaScript code into short and simple methods.

---

## 🎯 Purpose of jQuery

- Simplifies DOM manipulation
- Makes event handling easier
- Simplifies AJAX calls
- Reduces browser compatibility issues

---

## 🚀 Features of jQuery

- HTML / DOM Manipulation
- CSS Manipulation
- HTML Event Methods
- Effects and Animations
- AJAX
- Utility Functions

---
