JavaScript Content Methods - Notes

1. innerHTML

Gets or sets the HTML content inside an element. Parses HTML tags.

Example:
element.innerHTML = "<b>Hello</b>";
Output: Hello

2. innerText

Gets or sets the visible plain text inside an element. Ignores HTML tags. Slower than textContent.

Example:
element.innerText = "<b>Hello</b>";
Output: <b>Hello</b>

3. textContent

Gets or sets all the text in an element, including hidden text. Also ignores HTML tags. Faster than innerText.

Example:
element.textContent = "<b>Hello</b>";
Output: <b>Hello</b>

4. value

Gets or sets the value of form elements like <input>, <textarea>, or <select>.

Example:
let inputVal = document.getElementById("myInput").value;
