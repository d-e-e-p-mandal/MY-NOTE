## Redux :

State Management Library for JS Apps
Redux is built for larger, more complex applications.
Redux Toolkit is the official recommendation of writing Redux code.


### Store :
A centralized store holds the whole state tree of your application.

### Reducers :
Functions that take the current state and ans action as arguments, and return a new state result. In other words
state, action => new State.

### Action :
It is a plain JavaScript object that has a type field. (like event)

### Slice :
Collenction of Redux reducer logic and actions for singl together in a single file.
