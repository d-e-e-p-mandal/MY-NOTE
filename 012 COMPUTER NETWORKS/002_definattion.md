### Defination : Computer Network

A computer network a set of nodes connected by communication links.

A node can be a computer. printer or any other device capable of sending/ reciving data genrerated by other nodes in the network.

Example : computer, server, printer, security camera, etc.


### Commucincation Link :

A communication link can be a wired link or wireless link. The link carries the information.
- Wired : cable
- Wireless :
